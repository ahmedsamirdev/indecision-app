# Indecision App

Indecision is an app that let the computer guide you in your added options (todos).

### Tech stack

- HTML
- Sass
- JavaScript
- React


### **Features**

- Add option todo later.
- Remove one or all options.
- Capability to choose your next option (todo) randomly.
